# Dev Skill

Start the development server for ClaudeDesk.

## Instructions

When the user runs `/dev`:

1. **Start development server**
   ```bash
   npm run dev
   ```
   This runs both server and client in watch mode using concurrently.

2. **Inform the user**
   - Server will be at: http://localhost:8787
   - Hot reload is enabled for both server and client
   - Press Ctrl+C to stop

## Alternative Commands

- `/dev server` - Start only the server: `npm run dev:server`
- `/dev client` - Start only the client: `npm run dev:client`
